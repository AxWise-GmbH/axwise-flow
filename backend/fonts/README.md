# Font Files for PDF Generation

This directory contains font files used for PDF generation with Unicode support.

## DejaVu Sans

The DejaVu fonts are a font family based on the Vera Fonts. Its purpose is to provide a wider range of characters while maintaining the original look and feel.

To use Unicode characters in PDF exports, you need to download the DejaVu Sans font file:

1. Download DejaVuSans.ttf from: https://github.com/dejavu-fonts/dejavu-fonts/raw/master/ttf/DejaVuSans.ttf
2. Place it in this directory
